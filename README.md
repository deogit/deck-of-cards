# Stack of Cards

A PixiJS + TypeScript demo that shows five facedown playing cards on a casino-style table. Click a card to flip it and reveal a random card face.

## Requirements

- Node.js 20+
- npm

## Run Locally

1. Clone the repository:

```powershell
git clone <your-repo-url>
cd stack-of-cards
```

2. Install dependencies:

```powershell
npm install
```

3. Start the development server:

```powershell
npm run dev
```

4. Open the app in your browser:

```text
http://localhost:8080
```

If Vite does not open the browser automatically, paste the URL above manually.

## Build

To create a production build:

```powershell
npm run build
```

## Notes

- Click each facedown card to flip and reveal a unique random card.
- After all five cards are revealed, click again to start a new round.
